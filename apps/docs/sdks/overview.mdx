## Server-side SDKs

Dub offers a variety of server-side helper libraries to reduce the amount of work required to use Dub's REST APIs. Below are the installation instructions for these libraries in a variety of popular server-side programming languages:

<CodeGroup>

```bash TypeScript
npm install dub
```

```bash Go
go get github.com/dubinc/dub-go
```

```bash Python
pip install dub
```

</CodeGroup>

You can also reference the official documentation for each language to see how to leverage Dub's server-side SDKs in your applications:
